# Бот для управления расписанием трибун в Discord

## Описание
Этот бот предназначен для управления расписанием трибун в Discord-сервере. Он позволяет создавать расписание на неделю и записываться на различные части трибуны в определенные дни.

## Функциональность
- Создание расписания на неделю двух типов (команды `/week1` и `/week2`)
- Интерактивная запись пользователей на трибуны через кнопки
- Автоматическое обновление главного сообщения с расписанием
- Автоматическая очистка устаревших пользовательских сессий

## Установка и запуск

### Требования
- Node.js 16.6.0 или выше
